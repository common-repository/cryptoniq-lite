== Description ==

<h4 id="item-description__cryptoniq-payment-plugin-for-wordpress-decentralized">Cryptoniq | Cryptocurrency Payment Plugin for WordPress (Decentralized)</h4>

<p>Pay with <strong>BTC</strong>, <strong>ETH</strong>, <strong>TRX</strong>, <strong>DOGE</strong>. Shopping has never been so easy with cryptocurrencies.</p>

<p>Cryptoniq is a <strong>Woocommerce Payment Plugin</strong>, which allows to pay for products with <strong>cryptocurrencies</strong>. It was built with the vision to make the payments comfortable as much as possible.</p>

<h3 id="item-description__cryptoniq-coins">Coins</h3>

<p>Plugin supports <strong>ETH</strong> (For lite Version Only).</p>
<p>Get <a href="https://codecanyon.net/item/cryptoniq-cryptocurrency-payment-plugin-for-wordpress/22419379?rel=divengine" target="_blank"><strong>Pro</strong></a> version and use all Top coins:</p>

<ul>
	<li><strong>Bitcoin</strong> (BTC)</li>
	<li><strong>Ethereum</strong> (ETH)</li>
	<li><strong>Tron</strong> (TRX)</li>
	<li><strong>Dogecoin</strong> (DOGE)</li>
</ul>

<h3 id="item-description__cryptoniq-decentralized-payment-system">Decentralized Payment System</h3>

<p>As we know, most payment systems require registration on their services, creating wallets there - controlling your money. Also charging withdrawal fees from you.</p>

<p>Cryptoniq uses a <strong>Decentralized Payment Method</strong> for payments. You just need to add your wallet addresses to plugin options panel and user will send the money directly to your address. It gives the following advantages:

</p>

<ul>
    <li>Direct Payment</li>
    <li>No 3rd Party Service</li>
    <li>No Withdrawal Fees</li>
</ul>

<h3 id="item-description__cryptoniq-powerful-options-panel">Powerful Admin Panel</h3>

<p>Plugin has it's own Admin Panel, where you can set your options. It allows you to set wallet addresses, add coin prices for products, choose payment coins and much more...</p>

<h3 id="item-description__cryptoniq-payment-box">Payment Box</h3>

<p>There is a beautiful &amp; responsive payment box with an awesome design, compatible with all devices. Make your payment with one click.</p>

<h3 id="item-description__cryptoniq-currencies">Currencies</h3>
<p>Plugin supports 169 currencies (USD, EUR...)</p>

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Add post/page, click on 'Cryptoniq' button next to 'Add Media'.
4. Add images, select options and save it.

== Frequently Asked Questions ==

= What is Cryptoniq and why it's unique? =

Cryptniq is a fully decentralized payment plugin, which means there are no 3rd party services. Payment goes directly between buyer and seller.
